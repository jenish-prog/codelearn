## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually http://localhost:5173)

## Usage

- Write or paste JavaScript code in the code editor
- Click the "Execute" button to run your code
- Watch the execution visualization panel to see your code execute step-by-step
- Toggle between dark and light themes using the Settings button
